# Fonts Directory

Thư mục này chứa các font file hỗ trợ tiếng Việt cho việc tạo PDF chứng nhận.

## Cách thêm font:

1. Tải font hỗ trợ tiếng Việt (ví dụ: Arial, Times New Roman, hoặc font miễn phí như Roboto, Open Sans)
2. Đặt file font (.ttf) vào thư mục này
3. Đảm bảo file có tên: `arial.ttf`, `Arial.ttf`, `times.ttf`, hoặc `TimesNewRoman.ttf`

## Font miễn phí hỗ trợ tiếng Việt:

- **Roboto**: https://fonts.google.com/specimen/Roboto
- **Open Sans**: https://fonts.google.com/specimen/Open+Sans
- **Noto Sans**: https://fonts.google.com/specimen/Noto+Sans

Sau khi tải, đổi tên file thành `arial.ttf` hoặc `times.ttf` và đặt vào thư mục này.
